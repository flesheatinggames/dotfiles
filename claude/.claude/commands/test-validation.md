# Test Validation

<command>
    <instructions>
        Following your best practices, run the **complete** set of tests in the project. Resolve **all** failing tests. 100% test pass rate is **required**. 
    </instructions>
    <process>
        1. Use `debug-specialist` agent to run the **complete** set of test for the current code base. If no issues are found, go to step 6. If issues are found, go to step 2.
        2. For any issues that are found, use `debug-specialist` to diagnose them.
        3. Delegate to the appropriate agents to fix the errors based on the agent selection directive in your best practices.
        4. Use `karen` to verify that the work was actually done.
        5. Continue fixing errors until **100% pass rate** achieved.
        6. Prepare a report summarizing findings and actions and provide it to the user.
    </process>
</command>
