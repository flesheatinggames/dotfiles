    <instructions>
        Analyze the current project and identify opportunities to remove unneeded files and improve organization.
    </instructions>
    <process>
        Use `sequential-thinking` to:
        1. Analyze each file in the project, looking for files that are unneeded.

        2. Analyze the project structure and organization and look for opportunities to improve.

        4. If the user approves, carry out your recommendations in order to clean up the project.
    </process>
</command>
